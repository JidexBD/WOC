# Devx website Design 
##  Jadoo is a travel agency website 
    The project includes responsive UI components,
    a mobile-first navigation system, a right-side sliding sidebar, and clean
    semantic HTML/CSS architecture.

## Features
    Fully responsive navigation bar
    Modern layout using custom variables (CSS vars)
    Scalable component-based structure
    Optimized assets and typography
## Tech Stack
     HTML5 (semantic structuring)
    CSS3 (Flexbox, Custom Properties, Media Queries)
